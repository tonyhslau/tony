Gameplay Draft

This is the draft of the gameplay layout. Blue colour lines and dots are the “S” side and red colour dots are the “N’ side. These two sides have their tension to sticky their opposite side. That is “S” side will sticky “N” side but “S” will push “S” away, “N” will push “N” away.

In the beginning, the player will randomly stand inside the octagonal playground. Player need to controls the switch (Black colour rectangle) for changing the magnetic value of the wall. It can make the magnetic cube move away. Player ought to push magnetic cube into the exit hole(the circle with yellow and black colour) within the limited time for winning this game level.

In the game, player need to control the switch for the magnetic value using the number keyboard. When player presses the related number key, the related switch will change the colour from black to white, or white to black. It makes the magnetic cube(you) move away.
When player clicks the “?” button at GamePage Page, it will go to Instruction Page.

Controller Diagram

For the gameplay, player ought to use the number keyboard to control the switch for the magnetic value using the number keyboard.
For the basic control, player can use the mouse to select the menu options and help button. Player also can using arrow keys that is up and down keys to choose the option. Then, press the “Enter” key for windows OS, the “return” key for MacOS to confirm the option.
